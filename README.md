# 贸易企业出口退税规则判断助手

一个完全本地运行的 React + Vite 网页工具，根据 HSCode、企业类型、出口条件、进项凭证和海关监管状态，给出初步出口退税判断与风险提示。已导入官方出口退税率文库 2026B 版。

> 当前 MVP 仅支持外贸企业免退税初步判断，页面中的企业类型固定为“外贸企业”。

## 安装和启动

需要 Node.js 20.19 或更高版本。在本目录中执行：

```bash
npm install
npm run dev
```

终端会显示本地访问地址，默认通常为 `http://localhost:5173`。

其他命令：

```bash
npm test       # 运行判断引擎单元测试
npm run build  # 生成生产构建
npm run preview
```

## 数据维护

- `src/data/refund_rates_2026B.json`：由官方 2026B 版 DBF/FPT 文库转换，包含 11,727 条自 2026 年起有效的税率区间和 32,099 条商品名称。系统按官方基础商品码进行最长前缀匹配。
- `src/data/refund_rates_sample.json`：早期 MVP 样例数据，当前应用已不再读取。
- `src/data/dual_use_hs_codes_2026.json`：从商务部、海关总署2026年度《两用物项和技术进出口许可证管理目录》的出口目录中提取的636个10位参考海关商品编号。只用于HSCode初筛；是否受控必须继续核对管制清单、物项技术参数、用途、用户、目的地及临时管制公告。
- `src/data/legal_basis.json`：使用唯一 `id` 记录政策文件、条款、摘要、有效期和数据版本。更新条款时应保持 ID 稳定；已失效条目应更新状态和有效期，不要直接复用其 ID 表示新政策。
- `src/data/rules.json`：记录规则 ID、版本、启用状态、结果类型和关联政策 ID。判断逻辑集中在 `src/engine/evaluateRefund.js`，组件中不放置业务判断。

修改 JSON 后，请先执行 `npm test` 和 `npm run build`，再将更新后的数据用于实际判断。

## 出口管制初筛说明

输入10位HSCode后，页面会自动查询两用物项参考目录。命中时转入人工复核；未命中时也不代表已确认不受控。海关商品编号不是两用物项的最终判定依据，本功能不能替代出口管制物项识别、商务部咨询或许可申请。

## 样例验证

- `8479899990`：2026 年匹配退税率 13%。
- `8541430000`：2026 年 3 月 31 日前匹配 9%，4 月 1 日起匹配 0%。
- `8504319000`：匹配基础商品码 `85043190`，退税率 13%。

## 重要声明

本工具只用于内部初步判断，不构成税务、海关或法律意见，不能替代主管税务机关、海关、专业顾问或正式出口退税率文库的判断。政策变更后，应先更新规则库、政策依据库和退税率数据，再使用系统判断。
